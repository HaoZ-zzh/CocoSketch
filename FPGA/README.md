FPGA implementation of CocoSketch
============

File Description
--------------------
*  `crc32_64bit_gen.v`: responsable for generating hash values.
*  `CRC32_D64.v`: a submodule of crc32_64bit_gen.v
*  `CocoSketch.v`: the main implematation of CocoSketch.

Experimental setup
--------------------
* compile: Vivado 2020
* language: Verilog
